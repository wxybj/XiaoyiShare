## 一个德国银行账户，谁用谁知道

我不是在欧洲生活的人，但也搞了一个 N26 银行账户——因为它太特么好用了。

N26 是德国非常出名的数字银行，全线上运营，没有线下网点。在全球有超过 700 万用户。它在欧洲是正儿八经的持牌银行，享受欧盟存款保障。

## N26 能干什么

[N26](https://n26.com/r/qinyiw44408c) 主要解决两个问题：

1. **欧洲收款**：需要一个欧洲 IBAN 来收欧元
2. **全球消费**：它的万事达卡在欧洲和全球很多地方免手续费

我开它的契机就是：给欧洲客户做项目，客户只接受 IBAN 转账。中国内地银行没法直接收 SEPA 转账，这个需求卡了我好几周。

最后开了一个 N26，客户直接打欧元给我，到账快、手续费几乎为 0。

## 开户需要的条件

目前 N26 开户需要满足其中一个：
- 持有欧洲居留/工作签证
- 或者通过一些方式认证（非欧洲居民也可以，只是审核流程不同）

建议准备：
- **护照**
- **欧洲地址**（如果需要的做地址证明）
- **视频通话**：开户时需要跟客服视频认证

## 开户步骤

1. **下载 N26 App**
2. **注册并填写个人信息**
3. **视频身份验证**：跟客服视频，展示护照
4. **等待审核**（通常几分钟到几小时）
5. **激活账户**：收到虚拟卡就开好了
6. **申请实体卡**（可选）

**Q: N26 有中文吗？**

**Q: 账户有最低存款要求吗？**
A: N26 Standard（免费基础账户）没有最低余额要求。
