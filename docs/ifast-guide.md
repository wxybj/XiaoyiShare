## iFAST 是什么

[iFAST](https://www.ifastgb.com/tellafriend/qinyiw4955) 是一家总部在新加坡的金融科技公司，2017 年在港交所上市。它在英国有一个 iFAST Global Bank，持有英国 FCA 监管牌照。

iFAST 的核心价值是：**一个账户 = 英国银行账户 + 环球投资平台**。

- **银行部分**：持有英国 FCA 牌照，存款受 FSCS 保障（最高 85,000 英镑）
- **投资部分**：支持英美港新日等多国股票、ETF、基金
- **多币种**：英镑、美元、欧元、港币、新加坡元等

## 开户流程

1. 打开 [iFAST 官网](https://www.ifastgb.com/tellafriend/qinyiw4955) 申请开户
2. 填写个人信息（姓名、地址、职业等）
3. 身份认证：护照/身份证 + 地址证明
4. 视频认证（可选，有些情况需要）
5. 等待审核（一般 1-3 个工作日）
6. 开户成功，获得英国银行账号和投资权限

**需要的材料：**
- 身份证/护照
- 地址证明（水电费/银行账单，3个月内）
- 税务信息（中国税号）
