## 香港卡资金怎么衔接

如果你是去香港办卡顺便做资产配置，我比较建议按这个顺序来：

1. 先开通信银国际 GBA、汇丰 Pulse 等香港账户或信用卡相关账户
2. 确认自己账户里有可解释来源的港币资金
3. 通过邀请链接注册复兴证券，并完成实名审核
4. 用本人同名香港银行账户转入 HK$10,000 或 HK$20,000
5. 等入金确认后，再看活动中心是否满足奖励条件

不要用第三方账户代付，也不要为了奖励临时拼资金。券商和银行都看重姓名一致、资金来源清楚，这一步比“快”更重要。

## 风险提醒

证券账户不是银行存款，资金进入券商后如果参与股票、期权、新股认购，就会有市场波动、融资利息、汇率和流动性风险。新户奖励可以顺手拿，但别为了几百港币补贴去做自己看不懂的交易。
